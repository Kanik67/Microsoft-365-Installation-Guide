# Microsoft 365 Installation Guide

A simple step-by-step guide to install Microsoft 365 on Windows.

## Features
- Official installation method
- Easy to follow
- Beginner friendly

## Installation

1. Download Microsoft 365 from the official Microsoft website.
2. Run the installer as Administrator.
3. Wait for the installation to complete.
4. Sign in with your Microsoft account that has a valid Microsoft 365 subscription or license.
5. Open any Office app and verify activation.

## Requirements
- Windows 10 or Windows 11
- Internet connection
- Microsoft Account
- Valid Microsoft 365 license

## Disclaimer
This repository is intended for educational purposes and only documents the official installation process. It does not include unofficial activation methods.
